PASSWORD POLICY

1. What makes a strong password?

A strong password is long, unique, and difficult to guess. It should contain a combination of uppercase letters, lowercase letters, numbers, and special characters. Strong passwords reduce the risk of unauthorized access to accounts.

2. Why should passwords be unique?

Unique passwords ensure that if one account is compromised, other accounts remain secure. Using different passwords for each account prevents attackers from gaining access to multiple services with a single stolen password.

3. What is Multi-Factor Authentication (MFA)?

Multi-Factor Authentication (MFA) is a security method that requires users to provide two or more verification factors before gaining access to an account. Examples include a password combined with a mobile authentication code or biometric verification.

4. What are the risks of password reuse?

Password reuse increases security risks because if one website suffers a data breach, attackers can use the stolen password to access other accounts belonging to the same user. This attack is known as credential stuffing.

*Password Policy
1.Use passwords with at least 12 characters.
2.Include uppercase and lowercase letters.
3.Use numbers and special characters.
4.Avoid personal information such as names, birthdays, or phone numbers.
5.Use a unique password for every account.
6.Enable Multi-Factor Authentication whenever possible.
7.Change passwords immediately if a breach is suspected.
8.Store passwords securely using a password manager.
9.Never share passwords with others.
10.Regularly review and update important account passwords.
